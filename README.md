#Heading H2# CGPA Calculator

A simple and efficient CGPA (Cumulative Grade Point Average) calculator that allows students to calculate their GPA based on their grades and credit hours. This tool helps to easily compute and track academic performance over multiple semesters.

## Features

- Input grades and corresponding credit hours for each course.
- Calculate GPA for individual semesters.
- Calculate cumulative GPA (CGPA) based on multiple semesters.
- Supports both numerical and letter grade input (optional).
- Easy-to-use interface.

## Requirements

- Python 3.x
- No additional libraries required (core Python libraries only).
